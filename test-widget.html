üìù Instructions

  1. For local testing: Use http://localhost:8000 (or whatever port your server uses)
  2. For production: Replace with your production domain
  3. Replace your-site-here with your actual website URL (e.g., https://yourname.com)
  4. Choose the icon color that matches your website's theme

üöÄ Testing Locally

To test the webring locally, start a local server:

# Using Python 3
python3 -m http.server 8000

# Using Node.js (if you have http-server installed)
npx http-server -p 8000

# Using PHP
php -S localhost:8000

Then open http://localhost:8000 in your browser to see the main webring page, and http://localhost:8000/test-widget.html to see this test page.
